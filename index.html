WAKELAND TETRIS

ABOUT THE GAME

WAKELAND TETRIS IS AN OPEN-SOURCE BROWSER IMPLEMENTATION OF TETRIS BUILT WITH UNITY GAME ENGINE. IT AIMS TO BE A RIGOROUSLY-EFFICIENT AND, BY EXTENSION, FAST VERSION THEMED AFTER THE WAKELAND WOLVERINES OF WAKELAND HIGH SCHOOL.

HOW TO PLAY

YOUR OBJECT IS TO BUILD AS MANY LINES AS POSSIBLE WHILE PREVENTING BOARD OVERFLOW. ONCE THE BOARD IS FILLED, THE GAME IS OVER AND THE GAME RESETS.
